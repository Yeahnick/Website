Slot Machine

Mobile Game

One of my projects for my mobile development class was to program a slot machine using Kotlin with Android studio. This was a great opportunity to learn more about mobile features such as Graphical User Interface and Saving Data. We used libraries like Room for local database and Retrofit with Heroku for Cloud.

How does it Works?

The app has basic slot machine features like: multiple Bets, Images to represent the Wheel and returns with calculated odds.

-Using a "Radio Button" to select different bets we detect what the player selects.

-When the player Presses "Play" we verify if they have enough funds and then it spins the wheels randomly between 9 images.

-Then a multiplier is applied to the bet if 2 of 3 images are identical and a different one if 3 are identical instead.

-The password is used to add 100$ to your funs. Its only for development and is not an account related feature.

-If you select the "Check Box" : "Reckless" , it will change the algorityhm and the bets and profits are increased.

-We also store localy all the results and bets when clicking to play.

This project was simple and quick to create but still includes some great fundamentals like saving instances so that you are keeping the same page when you leave focus. Proper error handling so that the app doesn't crash if certain conditions were not met.

Additionally learned how to emulate different devices and test as if it were on mobile.
